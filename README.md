# History Navigator 

A browser history manager that actually uses a **Doubly Linked List** under the hood just like the same way real browsers work.

Built with Node.js, Express, MongoDB and React.

## What it does
- Visit URLs and track your history
- Go back and forward through pages
- Delete any history entry
- Data saved to MongoDB

## Tech Used
Node.js · Express · MongoDB · Next.js · Doubly Linked List

## Why I built this
Wanted to apply DSA in a real project instead of just solving leetcode problems.
